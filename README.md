# Portfolio — Bakka Sri Santosh Vihar
### AI & Machine Learning Engineer

A premium, production-quality personal portfolio website.

## Project Structure
```
portfolio-santosh/
├── index.html    — Main HTML (1306 lines, fully semantic)
├── styles.css    — Design system + all styles (~38KB)
├── main.js       — Interactions + animations (~18KB)
└── README.md     — This file
```

## Features
- ✅ Dark / Light mode with OS preference detection
- ✅ Sticky navigation with scroll spy
- ✅ Smooth scroll & section reveals
- ✅ Mouse-follow gradient effect
- ✅ Glassmorphism cards
- ✅ Animated counters
- ✅ Subtle card tilt on project cards
- ✅ Mobile-responsive with hamburger nav
- ✅ Contact form with validation
- ✅ Full accessibility (ARIA, keyboard navigation)
- ✅ SEO meta tags + Open Graph + Twitter Cards
- ✅ Inline SVG illustrations (no external image deps)
- ✅ Reduces motion for accessibility

## How to Open
Simply open `index.html` in any modern browser.
No build tools or servers required.

## To Deploy
Upload all three files to any static host:
- Vercel, Netlify, GitHub Pages, Cloudflare Pages

## Customization
- Update contact links in `index.html` (email, LinkedIn, GitHub)
- Replace resume `href="#"` with your actual PDF path
- Update certifications to match your actual certs
